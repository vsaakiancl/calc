# Fancy Scientific Calculator

## What This Is

A web-based scientific calculator with a polished glassmorphism UI. Built with React, it provides basic arithmetic and trigonometric functions with a scrollable calculation history panel. Designed to look and feel premium — frosted glass effects, blur, translucent panels.

## Core Value

Users can perform scientific calculations (especially trigonometry) in a beautiful, intuitive interface that feels delightful to use.

### Active

### Out of Scope

- Logarithms (ln, log10) — deferred to v2

- Math constants (π, e) — deferred to v2
- Desktop/mobile native app — web-first
- Real-time collaboration — not needed for a calculator
- User accounts or cloud sync — standalone tool
